Projeto_Idade

Este projeto foi desenvolvido em Python com o objetivo de praticar conceitos básicos da linguagem, como entrada e saída de dados, variáveis, operadores relacionais e estruturas condicionais.

O programa solicita a idade do usuário e informa se ele é maior ou menor de idade, permitindo colocar em prática a lógica de programação de forma simples e objetiva.

Durante o desenvolvimento, pude reforçar meus conhecimentos em Python, além de aprender mais sobre a organização de projetos e o uso do GitHub para armazenar e versionar meus códigos.

Objetivo
Praticar os fundamentos da linguagem Python.
Desenvolver o raciocínio lógico por meio de estruturas condicionais.
Aprender a organizar projetos e utilizar o GitHub.
Evoluir minhas habilidades como estudante de Desenvolvimento de Software.

Este projeto faz parte da minha trajetória de aprendizado e demonstra minha evolução na programação. Conforme avanço nos estudos, pretendo criar versões mais completas e aplicar novos conceitos da linguagem
